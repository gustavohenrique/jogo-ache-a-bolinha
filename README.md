Game of Shells
==============

[![Demo](https://github.com/gustavohenrique/jogo-ache-a-bolinha/raw/master/screenshot.png)](https://gustavohenrique.github.io/jogo-ache-a-bolinha)

### Technologies

* JQuery
* Mocha
* Chai
* Gulp
* Bower

### Setup

```shell
npm install
npm install -g bower
bower install
./node_modules/.bin/gulp watch
```

Para build:

```shell
./node_modules/.bin/gulp build
```
